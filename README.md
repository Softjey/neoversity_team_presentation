# Humans of Neoversity

Сайт-візитка команди. Чистий React + Vite, без UI-бібліотек і CSS-фреймворків.

```bash
npm install
npm run dev      # http://localhost:5173
npm run build    # tsc -b && vite build → dist/
npm run preview
```

## Деплой

Railway підхоплює `Dockerfile` автоматично (`railway.json` це фіксує явно).

1. New Project → Deploy from GitHub repo → `Softjey/neoversity_team_presentation`
2. Settings → Networking → **Generate Domain**
3. Готово. Healthcheck уже налаштований на `/healthz`.

Змінні оточення не потрібні: `PORT` Railway підставляє сам, nginx читає його через
`envsubst` із `docker/nginx.conf.template`. Локально дефолт — `8080`.

### Локальна перевірка образу

```bash
docker build -t neoversity-card .
docker run --rm -p 8080:8080 neoversity-card   # http://localhost:8080
# або з іншим портом, як це робить Railway:
docker run --rm -e PORT=3000 -p 3000:3000 neoversity-card
```

Образ — двоетапний: `node:24-alpine` збирає, `nginx:1.29-alpine` роздає статику (~27 МБ).
Хешовані файли з `/assets/` кешуються на рік як `immutable`, `index.html` — `no-store`,
щоб новий деплой одразу доїжджав до користувача.

## Де що редагувати

| Що | Файл |
|---|---|
| Профілі учасників (усі поля анкети, акцентні кольори, теги) | `src/data/team.ts` |
| Цифри, сильні сторони, цінності, цілі, спільне майбутнє | `src/data/content.ts` |
| Дизайн-токени (палітра, шрифти, радіуси, easing) | `src/index.css` |
| Фото | `public/avatars/*.png` |

Кожен учасник має свій акцентний колір (`accent`), який протягнутий через всю картку,
модалку, підсвітку курсора й лінію в секції «Ніхто з нас не йшов прямо».

Якщо фото немає або воно не завантажилось — `Avatar` показує градієнтну монограму,
тож сайт не ламається.

## Секції

1. **Hero** — кінетична типографіка, літери виїжджають каскадом
2. **Marquee** — стрічка технологій команди (пауза на hover)
3. **Stats** — цифри докручуються від 0, коли потрапляють у вʼюпорт
4. **Team** — картки з 3D-тілтом і підсвіткою за курсором → модалка з повною анкетою
5. **Paths** — «різноманітність досвіду»: 4 старти → одна точка сходження
6. **TeamDNA** — bento-грід: сильні сторони, спільні цілі, цінності, унікальність
7. **Future** — спільна відповідь на «де бачимо себе після магістратури»

## Інтерактив

- Кастомний курсор із контекстним підписом (`data-cursor` на елементі)
- Модалка: `Esc` — закрити, `←` / `→` — перемикати учасників
- Індикатор прогресу скролу вгорі
- Усе вимикається при `prefers-reduced-motion: reduce` і на тач-екранах
